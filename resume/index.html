소개

서울대학교 수학교육과에 재학 중인 1인 빌더입니다. 고등학교 때부터 직접 문제를 발견하고 웹·앱 서비스를 기획·구현해 왔으며, 지금은 AI 에이전트를 팀으로 삼아 기획·개발·콘텐츠·운영까지 혼자서 빠르게 굴립니다. 사교육에 편중된 교육 구조를 바꾸는 "수학의 듀오링고"를 만드는 것이 장기 목표이며, 좋은 아이디어보다 돌아가는 결과물로 증명하는 것을 더 중요하게 여깁니다.

AI를 팀원처럼

기획·개발·리서치·콘텐츠를 한 명 + AI 에이전트 팀으로 운영합니다.

Hermes Agent

미니PC에서 24시간 자율 실행되는 개인 AI 에이전트 플랫폼. 데이터 수집 → 분석 → LLM 해석 → 결과 전달까지 크론 파이프라인으로 무인 운영합니다. 미국장 트레이딩 어시스턴트와 ETF 리서치 자동화가 이 위에서 돌아갑니다.

Claude Code · Codex

코딩 에이전트와 짝을 이뤄 기획·디자인·MVP를 만듭니다. Vercel·Figma·Supabase·GitHub Actions로 출시 루프를 짧게 가져가, 아이디어를 며칠 안에 작동하는 서비스로 바꿉니다.
